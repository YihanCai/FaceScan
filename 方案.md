# 人脸检测项目方案

## 项目概述
基于计算机视觉和深度学习的人脸检测系统，可识别人脸位置并从静态图像、视频流或摄像头实时画面中检测人脸。

## 核心功能

### 1. 基础人脸检测
- 静态图片中的人脸检测（支持 jpg/png/bmp 等格式）
- 视频文件中的人脸检测
- 摄像头实时人脸检测
- 支持多人脸同时检测

### 2. 检测结果输出
- 在原图上绘制人脸框并保存/显示
- 输出人脸坐标信息（x, y, w, h）
- 支持批量图片处理
- 检测结果导出（JSON / CSV 格式）

### 3. 扩展功能（按需添加）
- 人脸关键点检测（眼睛、鼻子、嘴巴等 68 点/106 点）
- 人脸对齐
- 人脸裁剪与保存
- 人脸识别（人脸库比对，判断是谁）
- 表情识别
- 年龄/性别预测

## 技术选型

| 方案 | 优势 | 适用场景 |
|------|------|----------|
| **OpenCV Haar Cascade** | 轻量、离线、速度快 | 简单实时检测 |
| **OpenCV DNN + Caffe/TensorFlow 模型** | 精度较高，平衡性好 | 通用检测 |
| **MTCNN** | 精度高，含关键点 | 高质量检测 |
| **RetinaFace / SCRFD** | 精度最高，SOTA | 高精度要求 |
| **MediaPipe Face Detection** | 跨平台、轻量、GPU 加速 | 移动端/实时 |
| **YOLOv8-face** | 端到端、速度快 | 实时多人脸 |

### 推荐技术栈（渐进式）
1. **快速原型**：OpenCV Haar Cascade
2. **通用方案**：OpenCV DNN + ResNet SSD / YuNet
3. **生产方案**：MediaPipe 或 RetinaFace + ONNX Runtime
